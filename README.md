# Student Management System – JDBC + MySQL (VS Code)

A clean, layered **console-based Java project** to manage Full-Time and Part-Time students using **JDBC + MySQL**.

---

## ✅ Features

- Add Full-Time Student
- Add Part-Time Student
- Remove Student
- View Student by ID
- View All Students
- Sort by Date of Joining
- Sort by ID
- Sort by First Name
- Proper packages, DAO, interfaces, and custom exceptions

---

## ✅ Technologies

- Java (JDK 11+)
- MySQL
- JDBC
- VS Code
